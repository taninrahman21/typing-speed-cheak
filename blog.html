Defference between Local Storage and Session Storage?

SessionStorage is similar to localStorage.The difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends. Whenever a document is loaded in a particular tab in the browser, a unique page session gets created and assigned to that particular tab.

Defference between Global Scope and Block Scope?

Global Scope: A variable when declared outside the block or function scope and are accessible wherever we want. Block Scope: A variable when declared inside the if or switch conditions or inside for or while loops, are accessible within that particular condition or loop.

What is JavaScript Event Loop?

The event loop is a mechanism that allows JavaScript to perform non-blocking operations. When an asynchronous task is started, the event loop will start running. Once the task is completed, the event loop will again check for any other tasks that need to be performed.
